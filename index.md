# 📁 r2filebox - Share files quickly and keep privacy

[![](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/rosabellelimitless743/r2filebox/releases)

r2filebox allows users to share files and text snippets with others. The tool uses cloud technology to store your data securely while providing fast speeds for uploads and downloads. You can share large files without worrying about complex hosting setups or server management. The application creates short codes for your files, which makes sharing links easy with friends or colleagues.

## 🛠 Features

r2filebox offers several tools to manage your data:

* Anonymous sharing removes the need for user accounts.
* Extraction codes protect specific files from unwanted access.
* Resumable uploads allow you to pause and restart large transfers.
* Temporary storage options ensure your files vanish after a set time.
* Cloud-native storage provides high reliability for all your uploads.

## 🚀 Getting Started

Once you launch the app, you see a simple interface. You do not need to configure complex server settings. The app connects to the cloud automatically.

To share a file, drag your document into the main window of the app. The app shows a progress bar as the file uploads. Once the upload finishes, the app provides a unique link. You send this link to anyone you wish to share the file with. If you set an extraction code, tell the receiver that code as well. They enter this code on the website to unlock the download.

## 📦 Managing Uploads

You manage your active files through the history tab in the app. This tab shows a list of every file you uploaded. You can see how many people downloaded your files. If you decide to remove a file before the expiration date, click the delete button next to the file name. This removes the file from the cloud server immediately.

## ⚙️ Configuration Settings

The settings menu allows you to change how the app handles your files:

* Default Expiration: Set how long the app keeps your files before it deletes them.
* Notification Settings: Choose if you want the app to notify you when someone downloads your file.
* Upload Limits: Set the maximum file size for your uploads to stay within your personal cloud storage quotas.

## 🛡 Security and Privacy

Your data stays private. The application uses modern encryption methods to wrap your files during transit. When you upload data, it goes directly to secure storage locations. No one sees your files unless you share the specific link. The code you set keeps the gate locked for any visitor who lacks the secret key. 

## 🔧 Troubleshooting

If you encounter issues during installation or usage, check these common items:

* Verify your internet connection remains active.
* Check if your firewall blocks the application from connecting to the cloud.
* Ensure you have enough storage space on your primary hard drive.
* Restart the application if the file upload stalls.
* Update your Windows operating system to the latest version to ensure full compatibility.

If the application fails to launch, try a fresh installation. Uninstall the current version, restart your machine, and download the installer again from the main release page.
